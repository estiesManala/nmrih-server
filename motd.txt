Greetings! This server uses basic variables and official maps. Added 2 maps from the workshop and 1 enabled: nmo_southpole_v1, nms_evildeadcabin and nmo_toxtethdark.
Variables:
alltalk - is a shared chat for the living and the dead;
respawn near other players - respawn near players, from 100 to 150 units of distance (if there are respawn points), there are no respawn commands;
difficulty is "classic" - basic difficulty.
To restart the round / change the current map / change the next map, use the "End" key on the keyboard or via "Esc" and select "Start voting".
Added maps nmo from the workshop.
Good luck!
